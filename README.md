# vimconfig

This repo holds my neovim configuration (init.vim)


## Installation

Clone repo

```sh
# Unix
git clone git@github.com:MB3hel/vimconfig.git ~/.config/nvim

# Windows (powershell)
git clone git@github.com:MB3hel/vimconfig.git $Env:LOCALAPPDATA\nvim\
```

Update vim-plug

```sh
nvim +PlugUpgrade
```

Install plugins

```sh
nvim +PlugInstall
```
